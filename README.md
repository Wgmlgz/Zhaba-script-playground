# Zhaba-script-playground
Web playground for zhaba-script programming language 

## Build
- install [node](https://nodejs.org/en/) (17.x)
- install [emscripten](https://emscripten.org/docs/getting_started/downloads.html)
- `emcmake cmake .`
- `cmake --build .`
- `cd playground`
- `npm i`
- `npm run start`
